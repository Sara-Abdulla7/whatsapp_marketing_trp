Whatsapp Marketing With Odoo

Integrate whatsapp with odoo for marketing (ultramsg API Provider).

Overview

This module sends messages on WhatsApp via (Ultramsg API) to market, and it can also send media such as: images and files.

Configuration

    Go to Ultramsg API Ultramsg LINK
  • sign up in Ultramsg API
  • Create or ADD instance
  • Copy InstanceID and Token and API URL
  • Scan QrCode from Phone Device

Features

  • Allows you to send automatic messages via corn job.
  • Support Send MUlti Images And Files.
  • Support Grouping customers and then send them a group message.
  • Shows the status of message arrival.
